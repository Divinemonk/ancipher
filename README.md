# **Ancipher**
**"Alpha Numeric Cipher"**

<br> 

### 🪛 Installtion

It is a python (precisely v3) package, uploaded on [PyPi](https://pypi.org/project/ancipher/).

```
pip install ancipher
```

<br> 

### 📑 Usage
Firstly import it:  
```
from ancipher import anc
```
Now, use `anc()` (datatype: string)
```
anc("As simple as that!")
```
Output
```
45 51mpl3 45 7h47!
```

<br> 

### 🖱️ Requirements
Obviously ***Python 3***
